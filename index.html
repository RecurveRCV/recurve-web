START MINING RCV

Phase 1 — The Awakening is active. Your laptop can mine right now.

1 Install Dependencies

# Ubuntu/Debian
sudo apt install build-essential libssl-dev
sudo apt install libboost-all-dev libevent-dev

2 Clone and Build

git clone https://github.com/RecurveRCV/recurve-core
cd recurve-core
./autogen.sh
./configure
make -j$(nproc)

3 Start Mining

./src/recurved -daemon
./src/recurve-cli getmininginfo
./src/recurve-cli generatetoaddress 1 YOUR_RCV_ADDRESS

4 Check Balance

./src/recurve-cli getbalance
Mining difficulty is very low in Phase 1. This is intentional — early believers are rewarded. The Awakening belongs to retail.
